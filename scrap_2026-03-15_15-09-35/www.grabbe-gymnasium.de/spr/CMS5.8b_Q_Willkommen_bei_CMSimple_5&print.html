Die CMSimple Links

Wenn Sie ein originales oder modifiziertes CMSimple Template nutzen, belassen Sie bitte den Template Link im Template. Sie können diesen Link wie folgt erweitern:

  • Template by CMSimple - modified by Ihr Link

Wenn Sie ein anderes Template nutzen, setzen Sie bitte einen entsprechenden Link zu www.cmsimple.org in Ihrem Template, um die Popularität von CMSimple zu steigern, so geben Sie etwas zurück an das Projekt CMSimple.
